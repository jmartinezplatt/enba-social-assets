# Reporte semanal — Template

**Autor:** Bruno (Social Growth & Performance Director)
**Formato:** copiar y completar cada semana
**Frecuencia:** lunes de cada semana (cubre lunes anterior a domingo)

---

## Instrucciones de uso

1. Copiar este template a un archivo nuevo: `reporte-semana-N.md`
2. Completar cada seccion con datos reales
3. La columna "Estado" usa: ESCALAR / MANTENER / CORREGIR / CORTAR
4. La columna "Tendencia" usa flechas: ↑ sube, ↓ baja, → estable
5. Rellenar todos los campos. Si un dato no esta disponible, poner "N/D" y explicar por que.
6. El reporte se presenta a Team 4 (Marina, Franco, Bruno) y al usuario.
